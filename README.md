Hospital Management System:

A responsive web-based application built with Flask that provides real-time information about hospitals and doctors in Hyderabad. This project aims to simplify healthcare access by offering users updated, organized details on healthcare providers, including their specialties and contact information.

Features

Real-time Data Integration: Utilizes web scraping with BeautifulSoup to fetch live data from trusted healthcare directories.

Search & Filter: Users can search for doctors based on specialties or hospital names to find suitable care quickly.

Doctor & Hospital Directory: Displays categorized information with address, specialization, and contact info.

Responsive Design: Fully mobile-compatible and desktop-ready UI using HTML, CSS, and Bootstrap.

Database Support: Stores and manages data using SQLite, allowing efficient querying and updates.

Flask Backend: Handles routing, rendering templates, and connecting to the data layer seamlessly.
